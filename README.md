# ATM System (Python)

A menu-driven ATM system built using raw Python.

## Features
- Create account with validation
- Login with PIN (3 attempts)
- Deposit / Withdraw / Transfer
- Transaction history (last 5)
- Data persistence using file handling

## How to Run
```bash
python atm.py

## Notes
- Account number must be 10 digits
- PIN must be 4 digits
- Data is stored in atm.txt

## Learning Outcome
This project helped me understand input validation, control flow,
file handling, and real-world edge cases in Python.
